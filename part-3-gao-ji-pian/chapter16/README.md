图片不仅仅是为了展示美而存在，单纯意义上的美是美得不够长久的，只有富含深刻意义的美才会让人牢记，什么样的美才是具有深刻意义的美？能为我们带有信息，增强某些美好感觉，勾起美好回忆的美，这样的美就不在只是流于表象的美，因为它们在美的同时，还是信息的载体，很显然，我们在设计一个PPT时，PPT中插入的PPT不应是单纯以图片的美化程度来定，我们有更高的标准，那就是让观众觉得美，看得舒服。
